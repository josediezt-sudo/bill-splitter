# Divisor de Cuentas 🍽️

## Setup local
```bash
npm install
export ANTHROPIC_API_KEY=tu_clave
node server.js
```

## Deploy en Railway
1. Sube el código a GitHub
2. Crea proyecto en [railway.app](https://railway.app) → New Project → Deploy from GitHub
3. Agrega variable de entorno: `ANTHROPIC_API_KEY=tu_clave`
4. Railway asigna una URL pública automáticamente
